# MaktubLib
How to using with android studio.
### Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  ### on build.gradle .  Add the dependency
  
  	dependencies {
	        compile 'com.github.dvdinh307:MaktubLib:2.0'
	}
